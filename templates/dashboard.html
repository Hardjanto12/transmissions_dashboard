How to configure logs directory:
  1. Navigate to the folder containing your Transmission.log files
  2. Copy the full path (e.g., C:\logs\ or /var/log/transmission/)
  3. Paste the path in the "Logs Directory Path" field
  4. Click "Validate Directory" to verify the path
  5. Click "Save Settings" to apply changes
Tip: The directory should contain files named "Transmission.log" or "Transmission.log.1", etc.
